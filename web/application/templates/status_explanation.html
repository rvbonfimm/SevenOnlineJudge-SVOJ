Nós possuímos cinco tipos de resultados para os exercícios submetidos, sendo estes:



  • Status 1: Erro de sintaxe
  • Status 2: Resposta incorreta
  • Status 3: Tempo limite excedido
  • Status 4: Erro de apresentacao
  • Status 5: Codigo submetido com sucesso


Mas, o que significa, realmente, cada status?


Ok. Vamos lá a explicação:

  • Status 1

    • Status mais básico entre os outros. Neste, contém os erros mais comuns de programação. Erro de sintaxe, compilação, etc.


  • Status 2

    • Ocorre quando o resultado enviado pelo código do usuário não está igual ao código esperado pelo nosso sistema.


  • Status 3

    • Ocorre quando o código do usuário ultrapassa o tempo limite definido para a compilação desse código.


  • Status 4

    • Ocorre quando a resposta enviada pelo código do usuário contém formatações indevidas (espaços a mais, a menos, falta do caractér de retorno de linha).


  • Status 5

    • Status referente ao acerto do código enviado pelo usuário, pegando as entradas da maneira correta, gerando a(s) saída(s) esperada(s).
